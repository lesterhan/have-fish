# Epic: Import Page Redesign

Goal: Consolidate the import workflow so a user can go from a fresh CSV export to a fully imported set of transactions without leaving the page — including adding a new account and setting up its parser on the spot.

## Background

Currently the import flow requires jumping between pages: create an account on Settings, configure a parser on Settings, then come back to Import to actually upload. This is especially rough for a first-time import from a new bank. The redesign brings the add-account wizard and relevant parser management onto the Import page itself.
