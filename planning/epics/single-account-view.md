# Epic: Single Account View

A dedicated page for a single account that shows its transaction history,
running balance, and account-level metadata. This is the destination when the
user clicks an account row in the sidebar.

## Goals

- Route `/accounts/[id]` renders a page scoped to one account
- Shows the account name, type, currency, and current balance in a header
- Lists transactions for that account with a running balance column
- Entry point: sidebar account rows (wired up once this epic ships)
